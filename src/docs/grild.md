# Button 按钮

:::demo 通过`plain`属性可以设置为朴素的按钮；通过`disabled`属性可以设置按钮是否禁用；按钮的类型`type`有`info` 、`success`、`danger`、`warning`

```html
<se-row justify="space-between">
    <se-col :lg="8" :xs="24">ddd</se-col>
    <se-col :lg="8" :xs="24">ddd</se-col>
    <se-col :lg="8" :xs="24">ddd</se-col>
</se-row>
<se-row >
    <se-col :lg="8" :xs="24">ddd</se-col>
    <se-col :lg="8" :xs="24">ddd</se-col>
    <se-col :lg="8" :xs="24">ddd</se-col>
</se-row>
```
